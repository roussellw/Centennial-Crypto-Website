What is actually going on when we profit off of cryptocurrency mining?

Cryptocurrency mining is like doing work for someone else, but all the work is done automatically on a computer. Cryptocurrency was first proposed as a currency that could not suffer from inflation because there is a finite amount of it that can be produced. It was designed to be decentralized, so anyone can produce cryptocurrency by "mining" the cryptocurrency. This process involves using GPUs to hash blocks. Cryptocurrency is released when a certain amount of work has been done via hashing blocks. This is written into the fundamental source code of the cryptocurrency, and is commonly reffered to as "blockchain" technology.

Mining, the process of hashing blocks to earn cryptocurrency, uses an extrodinary amount of power. Also, the equiptment used in this process is expensive and requires matinence. Because of these drawbacks, most people are are deterred from doing this themselves as was orignially intended when the technology was invented. Thats where we come in! As mentioned, buying a share earns you a portion of a GPU used to mine cryptocurrency. Since you own the machine doing all the work you earn the cryptocurrency it generates. We do all the work when it comes to maintaining and insuring your machine. Because of this, we take 10% of the profit it generates. This offering is perpetual, meaning you get 90% of the profits generated for life. We are even willing to convert it to USD for for you using a third party service. You get your profits in a payout each month.
